## ðŸ“± APP: Kicks Shoes Mobile

**Kicks Shoes Mobile** is the companion mobile application that brings the shoe shopping experience directly to the user's fingertips. Built with React Native, the app provides a native look and feel for both iOS and Android, ensuring high performance and a consistent user experience. The app allows users to easily browse products, manage their accounts, and make purchases on the go.

Similar to the web version, the mobile app includes core functionalities such as login/registration, product viewing and searching, cart management, order tracking, and feedback submission. Future updates will focus on integrating push notifications for promotions and order updates, as well as mobile-specific features like image-based search.

### Technologies Used

- **Frontend:** React Native
- **Backend:** NodeJS, ExpressJS
- **Database:** MongoDB / PostgreSQL
